# 2017 Linguistics Holiday Party Survey Results

This is a Shiny app to summarize the results of the survey that we
(the [1st years][1years]) sent out to help with our decision about when
to organize the 2017 UMD Linguistics holiday party.

The results of the survey can be viewed [here][results].

[results]: https://shiny.adamliter.org/2017lingholidaypartysurveyresults
[1years]: http://ling.umd.edu/news/2017/aug/25/welcome-class-2022/
